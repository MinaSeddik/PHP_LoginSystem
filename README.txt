## Author
Mina Ibrahim Attia

## System Requirments
PHP, Mysql, Apache (or any other compatable web server)

## How to deploy the web site
1. Use the provided loginsystem.sql to create the database with the required table.
2. Modify the database configuration according to you setting ( server name, user name and password ) in ~\myproject\includes\config.php
3. Put the provided web site "myproject" into the web server. ( tested on Apache )
4. Start the server, then open any web browser to access the web site through http://localhost/myproject/ ( Assuming that you run the server on localhost, port 80)
